## Sampling Rate

- **After every task commit:** Run the per-task quick command from the per-task verification map below (~5s).
- **After every plan wave:** Run `pytest -m "not api_integration"` (~30s) followed by `pytest -m api_integration` (~3min).
- **Before `/gsd-verify-work`:** Full suite must be green AND per-recipe e2e (`test/e2e_inapp_v0_3.sh`) PASS for all 5 recipes.
- **Max feedback latency:** ~5s for per-task; ~3min for per-wave.
