<objective>
Build the **outbox pump** — a 100ms-tick asyncio task that publishes `agent_events` rows where `published=false` to Redis Pub/Sub channel `agent:inapp:<agent_instance_id>`. Per CONTEXT D-33/D-34: every event INSERT starts with `published=false`; the pump moves them to `published=true` after a successful Redis PUBLISH. SSE clients (Plan 22c.3-08) subscribe to these channels for live event delivery.

This is the **first transactional-outbox pattern in api_server** (PATTERNS.md "GREENFIELD"). RESEARCH §Pattern 3 has the source-of-truth code shape; this plan implements it.

Pitfall 3 strategy (per RESEARCH): **optimistic batch with single UPDATE on success**. If Redis fails mid-batch, the transaction rolls back; all rows stay `published=false`; next tick retries the entire batch. Possible duplicate publish (rows that DID succeed before failure get re-published). SSE clients dedupe by `agent_events.id` (D-10 idempotent client-side handling).

Per D-35 abandon-after-1h: events older than 1h with published=false stay unpublished; Last-Event-Id replay handles them via direct PG query, bypassing Redis.

Purpose: At-least-once delivery from PG-of-record to Redis Pub/Sub, with crash resilience.
Output: outbox pump service + 8+ integration tests using testcontainers-redis.
</objective>

**File 1: `api_server/src/api_server/services/inapp_outbox.py`** — paste-and-adapt RESEARCH §Pattern 3:

```python
"""Phase 22c.3 — outbox pump (Postgres → Redis Pub/Sub).

D-33/D-34: every agent_events row inserted with published=false; this
pump publishes to redis channel `agent:inapp:<agent_instance_id>` then
marks published=true. 100ms tick cadence (sub-second event delivery to
SSE clients in the happy path).

Strategy (Pitfall 3 strategy 2): optimistic batch with single UPDATE on
success. Iterate publishes; if any redis.publish fails, abort the batch
(transaction rolls back; no UPDATE runs); next tick retries from
scratch. Possible duplicate publish for rows that succeeded before the
failure — SSE clients dedupe by agent_events.id (D-10).

D-35 abandon-after-1h: rows older than 1 hour with published=false are
FILTERED out of the SELECT and stay published=false. Clients still
receive them via Last-Event-Id replay against agent_events directly
(Plan 22c.3-08), bypassing Redis.

D-32 multi-replica safety: FOR UPDATE SKIP LOCKED.
"""
from __future__ import annotations
import asyncio
import json
import logging
from datetime import timedelta

import redis.asyncio as redis_async

_log = logging.getLogger("api_server.inapp_outbox")

PUMP_TICK_S = 0.1            # D-34
PUMP_IDLE_TICK_S = 0.5
PUMP_BATCH_LIMIT = 100       # D-34
ABANDON_AFTER = timedelta(hours=1)   # D-35


async def outbox_pump_loop(app_state, stop_event: asyncio.Event) -> None:
    """Lifespan-managed pump. Runs until stop_event is set."""
    pool = app_state.db
    redis_client: redis_async.Redis = app_state.redis
    while not stop_event.is_set():
        try:
            published_n = await _pump_once(pool, redis_client)
        except Exception:
            _log.exception("phase22c3.outbox.pump_unexpected")
            published_n = 0

        if published_n == 0:
            try:
                await asyncio.wait_for(stop_event.wait(), PUMP_IDLE_TICK_S)
            except asyncio.TimeoutError:
                pass
        else:
            try:
                await asyncio.wait_for(stop_event.wait(), PUMP_TICK_S)
            except asyncio.TimeoutError:
                pass


async def _pump_once(pool, redis_client) -> int:
    """Pick up to PUMP_BATCH_LIMIT unpublished events; publish; mark.

    On any redis.RedisError mid-batch: abort (no UPDATE runs; rows stay
    published=false). Returns the count of rows successfully marked.
    """
    async with pool.acquire() as conn:
        async with conn.transaction():
            rows = await conn.fetch(
                """
                SELECT e.id, e.seq, e.kind, e.payload, e.correlation_id,
                       e.ts, c.agent_instance_id
                FROM agent_events e
                JOIN agent_containers c ON c.id = e.agent_container_id
                WHERE e.published = FALSE
                  AND e.ts > NOW() - $1::interval
                ORDER BY e.id
                FOR UPDATE OF e SKIP LOCKED
                LIMIT $2
                """,
                ABANDON_AFTER, PUMP_BATCH_LIMIT,
            )
            if not rows:
                return 0
            published_ids: list[int] = []
            for r in rows:
                payload = r["payload"]
                if isinstance(payload, str):
                    payload = json.loads(payload)
                channel = f"agent:inapp:{r['agent_instance_id']}"
                msg = json.dumps({
                    "seq": int(r["seq"]),
                    "kind": r["kind"],
                    "payload": payload,
                    "correlation_id": r["correlation_id"],
                    "ts": r["ts"].isoformat(),
                })
                try:
                    await redis_client.publish(channel, msg)
                except redis_async.RedisError:
                    _log.exception(
                        "phase22c3.outbox.redis_publish_failed",
                        extra={"event_id": int(r["id"])},
                    )
                    # Abort batch — UPDATE never runs; tx rolls back.
                    return 0
                published_ids.append(int(r["id"]))
            if published_ids:
                await conn.execute(
                    """
                    UPDATE agent_events
                    SET published = TRUE
                    WHERE id = ANY($1::bigint[])
                    """,
                    published_ids,
                )
            return len(published_ids)
```

    async with redis_client.pubsub() as pubsub:
        await pubsub.subscribe(channel)
        collected = []
        end = asyncio.get_event_loop().time() + timeout
        while len(collected) < n and asyncio.get_event_loop().time() < end:
            msg = await pubsub.get_message(ignore_subscribe_messages=True, timeout=0.5)
            if msg is None:
                continue
            data = msg["data"]
            if isinstance(data, (bytes, bytearray)):
                data = data.decode("utf-8")
            collected.append(json.loads(data))
        return collected
