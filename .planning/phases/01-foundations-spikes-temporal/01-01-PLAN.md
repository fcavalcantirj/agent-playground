<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client -> API | HTTP requests from browser to Go API; untrusted input crosses here |
| cookie -> session lookup | Signed cookie value used to look up server-side session; HMAC is the trust boundary |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-01 | Spoofing | Session cookie | mitigate | HMAC-SHA256 signing of session token prevents forging; cookie is HttpOnly + SameSite=Lax (D-09) |
