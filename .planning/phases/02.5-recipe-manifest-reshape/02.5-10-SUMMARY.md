# Phase 02.5 Plan 10: Gate A smoke-test matrix Summary

**One-liner:** Shipped `test/smoke-matrix.sh` + `make smoke-test-matrix` — the 2x2 (aider, picoclaw) x (anthropic, openrouter) runner that probes `whoareyou`, enforces D-47 pass criteria in-script, and leaves zero dangling `playground-*` containers.

## What Shipped

### `test/smoke-matrix.sh` (new, +300 lines, chmod +x)

Gate A runner that:

1. **Teardown guards.** Runs `docker ps -aq --filter 'name=playground-' | xargs docker rm -f` on entry AND via an `EXIT` trap (covers normal termination, `exit 1`, Ctrl-C, and shell errors). Honors the global CLAUDE.md "kill previous before starting" rule as T-02.5-06 mitigation.
2. **Healthcheck.** `curl -fsS --max-time 5 $API_URL/healthz` — fails fast with a clear operator hint if the server is not up.
3. **Dev auth.** `POST /api/dev/login` into a mktemp cookie jar; the jar is cleaned up by the same `EXIT` trap.
4. **BYOK env detection.** Reads `AP_DEV_BYOK_KEY` (Anthropic) and `AP_DEV_OPENROUTER_KEY` (OpenRouter). Missing keys cause cells to **SKIP**, not FAIL — CI without keys still passes.
5. **Recipe introspection.** For each cell, `GET /api/recipes/$recipe` is fetched once, and jq filters on `.providers[]` + `.models[]` decide:
   - Recipe doesn't declare provider → SKIP
   - Recipe has no `.models[]` entry where `provider == $p` → SKIP
   - Otherwise, picks `[.models[] | select(.provider == $p)][0].id` (D-48: first-match, cost-ordered).
6. **Cell execution.**
   - `POST /api/sessions` with `{recipe, provider, model}` → capture `.id`
   - `POST /api/sessions/$id/message` with `{text: "whoareyou"}` → extract `.text // .message // .reply`
   - Assert no top-level `.error` key in the response envelope (D-49)
   - `DELETE /api/sessions/$id`
   - Re-assert zero `playground-*` containers (T-02.5-06 belt-and-suspenders)
7. **Summary table + D-47 enforcement.** After the 2x2 loop:
   - Prints the full matrix (PASS/FAIL/SKIP per cell) + `PASS=n FAIL=m SKIP=k` count
   - `check_d47()` verifies: each recipe passes on ≥1 provider, ≥1 recipe passes on **both** providers (OpenRouter wiring proven), zero non-SKIP failures
   - `=== Gate A PASSED ===` (green) with `exit 0`, or `=== Gate A FAILED ===` (red) with `exit 1`

`set -uo pipefail` (not `-e`) is used intentionally so one failing cell never prevents the rest of the matrix from running.
