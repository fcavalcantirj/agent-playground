```go
func (l *limitedWriter) Write(p []byte) (int, error) {
    if l.n+len(p) > l.max {
        return 0, fmt.Errorf("%w: %d bytes", ErrTemplateSize, l.max)
    }
    n, err := l.w.Write(p)
    l.n += n
    return n, err
}
```

9. The goroutine pattern (cannot cancel `template.Execute` mid-run):
```go
ctx, cancel := context.WithTimeout(ctx, templateMaxDuration)

}
ch := make(chan result, 1)
go func() {

    lw := &limitedWriter{w: &buf, max: templateMaxBytes}
    err := tmpl.Execute(lw, data)
    ch <- result{s: buf.String(), err: err}

select {

    return res.s, nil
}
```

<threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| `agents/<id>/templates/*.tmpl` on disk → template engine | Template body is trusted code-review surface in v0.1. Phase 8 bootstrap writes into `agents/community/` — that content is untrusted and the closed FuncMap must defeat every known SSTI vector. |
| `data` parameter passed into Render → rendered output | Data includes resolved secrets. Output is written to a container's tmpfs `/run/secrets` — never to host disk, never to logs. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02.5-03 | Tampering / Elevation of Privilege | `text/template` renderer SSTI | mitigate | Closed FuncMap with 5 string-only functions (`default`, `quote`, `lower`, `upper`, `trim`). No `exec`, no `include`, no `printf`, no `js`, no `html`, no `readFile`, no `env`. Every dangerous FuncMap entry from sprig is absent. `Option("missingkey=error")` ensures typos don't silently resolve to empty strings. Rejected alternative (sprig) explicitly documented as "not imported" in code comment. Citation: oligo.security SSTI research — text/template without a user-controlled FuncMap is safe-by-default. |
| T-02.5-03a | Tampering | Path traversal via `template` field in recipe.yaml | mitigate | Regex `^[a-z0-9][a-z0-9_-]*\.tmpl$` rejects uppercase, path separators, dots (other than the required `.tmpl` suffix), and special characters. `os.Lstat` + `ModeSymlink` check rejects symlinks. `filepath.Abs` prefix check rejects resolved paths that escape `agents/<id>/templates/`. All three checks run in order; each is a belt-and-suspenders layer. |
| T-02.5-06 | Denial of Service | Infinite loop template or huge output | mitigate | `context.WithTimeout(ctx, 5*time.Second)` bounds wall-clock render time. `limitedWriter` with 64 KiB cap short-circuits bytes-written attacks. Goroutine pattern is required because `template.Execute` cannot be cancelled mid-run — the select returns ErrTemplateTimeout as soon as ctx expires; the goroutine leaks on malicious templates but the trust model (v0.1 = code-reviewed repo contents) accepts this. Phase 8 will add a separate syscall-level watchdog for community templates. |
| T-02.5-02 | Information Disclosure | Rendered secret leaks into logs or host disk | mitigate | Render returns a `string`; the caller (Plan 05 materialize) is responsible for routing it to container tmpfs only. Render itself never logs `data` — enforced by grep gate in the acceptance criteria (`grep 'logger.*data' template.go` must return 0). Rendered bodies are not cached; only parsed templates are. |
