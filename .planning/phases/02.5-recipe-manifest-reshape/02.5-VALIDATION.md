## Sampling Rate

- **After every task commit:** Run the package-scoped quick command (affected package only)
- **After every plan wave:** Run `cd api && go test ./... -count=1`
- **Before `/gsd-verify-work`:** `make test && make smoke-test-matrix` must be green; Gate B (`make test-architectural-drop-in`) green once
- **Max feedback latency:** 45 seconds (package-scoped); 180s (full + matrix)
